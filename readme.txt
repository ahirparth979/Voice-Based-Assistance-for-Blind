-create a new project directory and paste all files and directory in it.

-open terminal in the project directory to create a new environment.

-install virtualenv using pip
	pip install virtualenv

-create a new virtual environment in the project folder
	python -m venv virtual

-activate the virtual environment by running activate.bat
	virtual\Scripts\activae.bat

-install the requirements file
	pip install -r requirements.txt

-install the Pyaudio .whl package
	pip install PyAudio-0.2.11-cp38-cp38-win32.whl

-now your virtual environment is set.
